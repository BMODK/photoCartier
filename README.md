## 🌟 Sobre o Projeto

<div align="center">

> *"Cada lugar tem uma luz única. PhotoCartier ajuda você a encontrá-la."*

</div>

**PhotoCartier** é uma plataforma social e API inteligente criada **para fotógrafos, por fotógrafos**. Combinando descoberta de locais, gamificação e inteligência coletiva da comunidade, o PhotoCartier redefine como os fotógrafos brasileiros exploram, avaliam e compartilham os melhores spots fotográficos do país.

Mais do que um simples catálogo de lugares — o PhotoCartier é um **ecossistema vivo**, onde cada avaliação, cada badge conquistada e cada localização descoberta enriquece a experiência de toda a comunidade.

<br/>

<div align="center">

| 📍 Locais Mapeados | 📸 Fotógrafos | ⭐ Avaliações | 🏆 Rankings |
|:-:|:-:|:-:|:-:|
| **+12.000** | **+50.000** | **+300.000** | **Nacionais & Regionais** |

## 🎯 O Problema que Resolvemos

Fotógrafos brasileiros enfrentam diariamente desafios que custam tempo, dinheiro e oportunidades únicas:

```
❌  Sem saber a melhor hora para fotografar um local
❌  Sem informações sobre dificuldade de acesso
❌  Sem avaliações confiáveis de outros fotógrafos
❌  Sem ranking de qualidade para comparar locais
❌  Sem comunidade especializada e engajada
❌  Sem incentivo para explorar novos territórios
```

**PhotoCartier resolve tudo isso em uma única plataforma:**

```
✅  Horários ideais por local, época e condição de luz
✅  Avaliações detalhadas da comunidade fotográfica
✅  Ranking inteligente baseado em múltiplos critérios
✅  Sistema de gamificação que recompensa a exploração
✅  Perfis profissionais e reputação verificada
✅  Comunidade ativa e engajada em todo o Brasil
```

## 🏆 Como Funciona o Ranking

O sistema de ranking do PhotoCartier combina múltiplos sinais para garantir precisão e justiça:

```
                    ┌─────────────────────────────┐
                    │     SCORE DE UM SPOT         │
                    │                              │
                    │  Qualidade Visual   35%  ████│
                    │  Nº de Avaliações   20%  ██  │
                    │  Acessibilidade     15%  █▌  │
                    │  Diversidade Fotos  15%  █▌  │
                    │  Atualidade         10%  █   │
                    │  Engajamento         5%  ▌   │
                    └─────────────────────────────┘
```

### 📐 Fórmula de Ranking

```
Score = (Q × 0.35) + (A × 0.20) + (Ac × 0.15) + (D × 0.15) + (T × 0.10) + (E × 0.05)
```

| Variável | Significado |
|:-:|---|
| `Q` | Qualidade Visual média (1–10, baseada nas avaliações) |
| `A` | Peso normalizado do número de avaliações |
| `Ac` | Score de acessibilidade (trilha, estrada, infraestrutura) |
| `D` | Diversidade de tipos de fotografia no local |
| `T` | Atualidade das avaliações (decaimento temporal) |
| `E` | Taxa de engajamento (curtidas, saves, comentários) |

### 🗺️ Rankings Disponíveis

- 🇧🇷 **Ranking Nacional** — Top 100 spots do Brasil
- 🏙️ **Ranking Estadual** — Top 20 por estado
- 🌊 **Rankings por Bioma** — Amazônia, Cerrado, Pantanal, Mata Atlântica, Caatinga, Pampa
- 📷 **Rankings por Categoria** — Paisagem, Fauna, Astro, Urbano etc.
- 🌅 **Rankings Sazonais** — Spots da estação (seco, chuva, outono, verão)

## 🚀 Instalação

### ✅ Pré-requisitos

```bash
node --version   # >= 20.x
npm --version    # >= 10.x
docker --version # >= 24.x
```

### 1️⃣ Clone o Repositório

```bash
git clone https://github.com/seu-usuario/photocartier.git
cd photocartier
```

### 2️⃣ Instale as Dependências

```bash
npm install
```

### 3️⃣ Configure as Variáveis de Ambiente

```bash
cp .env.example .env
```

Abra `.env` e configure:

```env
# ── Banco de Dados ─────────────────────────────────────
DATABASE_URL="postgresql://user:password@localhost:5432/photocartier_db"

# ── Redis ──────────────────────────────────────────────
REDIS_URL="redis://localhost:6379"

# ── JWT ────────────────────────────────────────────────
JWT_SECRET="seu_secret_super_seguro_aqui"
JWT_EXPIRES_IN="7d"
JWT_REFRESH_EXPIRES_IN="30d"

# ── AWS ────────────────────────────────────────────────
AWS_ACCESS_KEY_ID="sua_access_key"
AWS_SECRET_ACCESS_KEY="seu_secret"
AWS_REGION="sa-east-1"
AWS_S3_BUCKET="photocartier-media"

# ── Google Maps API ────────────────────────────────────
GOOGLE_MAPS_API_KEY="sua_api_key"

# ── App ────────────────────────────────────────────────
PORT=3000
NODE_ENV=development
APP_URL="http://localhost:3000"
```

### 4️⃣ Suba o Banco de Dados com Docker

```bash
docker-compose up -d postgres redis
```

### 5️⃣ Execute as Migrations

```bash
npx prisma migrate dev
npx prisma db seed
```

## ▶️ Como Executar

### 🔧 Ambiente de Desenvolvimento

```bash
npm run start:dev
```

### 🐳 Com Docker Compose (recomendado)

```bash
# Subir toda a stack
docker-compose up -d

# Ver logs
docker-compose logs -f api

# Parar tudo
docker-compose down
```

### 🏭 Produção

```bash
npm run build
npm run start:prod
```

### 🧪 Testes

```bash
# Unitários
npm run test

# E2E
npm run test:e2e

# Cobertura
npm run test:cov
```

> 🔗 Após iniciar, acesse:
> - **API**: `http://localhost:3000`
> - **Swagger UI**: `http://localhost:3000/api/docs`
> - **Health Check**: `http://localhost:3000/health`
